QEMU安装踩坑日记

Posted on 2023-04-22，4 min read

安装编译

官网下载链接：点击此处

  • linux安装步骤
wget https://download.qemu.org/qemu-8.0.0.tar.xz
tar xvJf qemu-8.0.0.tar.xz
cd qemu-8.0.0
./configure
make
make install

常用排坑小技巧

  1. 直接输入cgcc会提示解决方案
  2. 可以直接看命令运行的最后的结果，会有错误提示百度搜索即可

安装系统要求（建议）

硬盘存储40GB（尽量大了好，不然编译过程中会出现磁盘root空间不足）
处理器：2个（建议大于等于）
内存：2GB（建议大于等于）
提示：如果存储硬盘不足，可以用参考这个博客 ubuntu 自带Disks工具进行磁盘分区操作
